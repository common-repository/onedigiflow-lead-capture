== Description ==

OneDigiFlow lead capture plugin allows users to automatically save contacts and deals into OneDigiFlow CRM.

= Main Features =

* Integration with OneDigiFlow
* Pull product SKU information to display as dropdown value in the contact form
* Get contact details directly into your Wordpress backend and OneDigiFlow CRM
* Provide custom CSS for the contact form to match your theme
* Select which product SKUs you want to display on the contact form
* Embed the form with shortcode on any page or post
* 100% Multi language

== Installation ==

Following the steps below, you can install plugin and running in seconds.

1. Upload 'onedigiflow-lead-capture' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
